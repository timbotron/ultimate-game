## Ultimate Game

##### Author: Tim Habersack

##### Date: 01/07/2015

Ultimate game is a command-line game that plays rock, paper, scissors, lizard, Spock. It is silly.

To play, run: 

`php ultimategame.php rock`

Or whichever weapon you prefer. Good luck!
