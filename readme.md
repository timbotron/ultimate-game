## Ultimate Game

##### Author: Tim Habersack

##### Date: 01/09/2015

Ultimate game is a command-line game that plays rps-15. It is silly.

http://www.umop.com/rps15.htm

To play, run: 

`php ultimategame.php rock`

Or whichever weapon you prefer. Good luck!
